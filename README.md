# Medical Cost Prediction - Linear Regression

## Descripción
Proyecto de Machine Learning que predice costos médicos anuales individuales utilizando Regresión Lineal. Desarrollado como parte de un portafolio profesional de Data Science.

## Objetivo
Construir un modelo interpretable que permita a la aseguradora "MediCare Plus" calcular primas personalizadas basadas en factores de riesgo individuales.

## Problema de Negocio
Actualmente, "MediCare Plus" utiliza un sistema de precios genérico que no refleja adecuadamente los factores de riesgo individuales, resultando en pérdidas para la empresa y primas injustas para clientes de bajo riesgo.

## Tecnologías
- Python 3.9+
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-Learn
- Joblib
- Jupyter Notebook
